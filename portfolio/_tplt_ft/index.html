My Contribution

Building a Lightweight, Scalable Reporting System

I implemented a fast, low-cost solution using Jotform + Google Sheets + Looker Studio to register, organize, and visualize field data (visits, photos, GPS location, products, comments). The system was designed to be scalable and easy to use by the promoters in real time.

Outcome / Key Learnings
  • A real-time map of field activities was created for the first time.
  • Clear visibility of actions by promoter, location, and product.
  • Organized visual history of POS (Point of Sale) execution, ready to be used by the marketing and sales teams.
